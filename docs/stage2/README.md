# Этап 2: Конфигурация

## Требования
1. **Параметры командной строки:**
   - `--vfs-path` — путь к VFS
   - `--log-file` — путь к лог-файлу
   - `--start-script` — путь к стартовому скрипту
   - `--config` — путь к конфигурационному файлу YAML

2. **Конфигурационный файл YAML** содержит те же параметры

3. **Приоритеты:** командная строка > конфигурационный файл

4. **Логирование** в JSON-формате с сообщением об ошибке

5. **Стартовый скрипт:**
   - Выполняет команды последовательно
   - Пропускает ошибочные строки
   - Отображает ввод и вывод

6. **Тестовые скрипты** для проверки всех параметров

## Реализация

### Конфигурация
- Класс `Config` в `src/config.py`
- Парсинг аргументов: `argparse`
- Загрузка YAML: библиотека `PyYAML`

### Логирование
- Класс `Logger` в `src/logger.py`
- Формат JSON с полями:
  ```json
  {
    "timestamp": "2024-01-15T10:30:00",
    "command": "ls",
    "args": ["-la"],
    "error": "Файл не найден"
  }
