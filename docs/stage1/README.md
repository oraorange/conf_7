# Этап 1: REPL

## Требования
1. Приложение в форме консольного интерфейса (CLI)
2. Приглашение к вводу должно содержать имя VFS
3. Парсер с раскрытием переменных окружения (например, $HOME)
4. Сообщение об ошибке выполнения команд
5. Команды-заглушки: ls, cd
6. Команда exit
7. Демонстрация работы в интерактивном режиме

## Реализация
- Основной код: `../src/emulator.py`
- Приглашение: `username@hostname:vfs_name$`
- Парсер: функция `parse_input()` с поддержкой $VARS
- Обработка ошибок: функция `handle_error()`
